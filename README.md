# Lascivious Vibrations

This is a simple web application that generates creative prompts and autopilot content based on selected 'drops' and stylistic elements.

## Features

- **Drop Selection**: Choose from different thematic 'drops' to influence prompt generation.
- **Prompt Generation**: Generates unique prompts by combining base themes with cinematic style elements.
- **Prompt Remix**: Enhances the current prompt with additional details.
- **Copy Prompt**: Easily copy the generated prompt to your clipboard.
- **Autopilot Content**: Generates sample blog, Pinterest, and video content based on the selected drop.

## Usage

Open `index.html` in your web browser. Select a 'drop' and use the 'Generate', 'Remix', and 'Copy' buttons for prompts. Use 'Generate Content' for autopilot content.

## Development Notes

- The application is built with pure HTML, CSS, and JavaScript.
- CSS is externalized into `style.css`.
- JavaScript functions handle dynamic content generation and user interactions.
- The 'Unlock Premium' button links to a Gumroad page for full drops.

## Deployment

This is a static site and can be deployed on any web server. For HTTPS support, deploy to platforms like GitHub Pages, Netlify, Vercel, or configure your web server with an SSL certificate.
